# Vue Pokemon

> A simple curated list of pokemons

## Run Application
`npm run dev`

## Built With

* [Nuxt JS](https://nuxtjs.org/)
* [Vuetify](https://vuetifyjs.com/en/)
* [Pokemon API](https://pokemontcg.io/)
